El bucle for

Un bucle for repite un bloque de código para cada elemento de una secuencia. Con range(n) puedes repetir n veces. Con for x in lista: recorres cada elemento de una lista.

💡 range() range(0, 10, 2) genera los números 0, 2, 4, 6, 8 — inicio, fin (exclusivo) y paso.
⏳

El bucle while

Un bucle while repite el código mientras una condición sea verdadera. Es ideal cuando no sabes cuántas veces necesitas repetir. ¡Cuidado! Si la condición nunca se hace False, el bucle es infinito.

🛑 break y continue break detiene el bucle inmediatamente. continue salta a la siguiente iteración sin ejecutar el resto.
🧠

¿Cuándo usar cada uno?

Usa for cuando sabes cuántas iteraciones necesitas o tienes una colección que recorrer. Usa while cuando la repetición depende de una condición que cambia durante la ejecución.

⚡ Enumerate for i, elem in enumerate(lista): te da el índice y el elemento a la vez. ¡Muy útil!
