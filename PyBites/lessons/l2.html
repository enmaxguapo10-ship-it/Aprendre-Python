¿Qué es una variable?

Una variable es como una caja con nombre donde guardas información. En Python, crear una variable es tan simple como escribir nombre = valor. No necesitas declarar el tipo — Python lo detecta solo.

💡 Dato clave Python usa tipado dinámico: una misma variable puede guardar un número y después un texto.
🔢

Tipos de datos básicos

Python tiene varios tipos fundamentales: int (números enteros como 42), float (decimales como 3.14), str (texto como "hola"), y bool (True o False). Cada uno sirve para cosas distintas.

🔍 Truco Usa type(variable) para saber qué tipo tiene cualquier variable en Python.
✏️

Reglas de nombres

Los nombres de variables deben empezar por una letra o guion bajo, no pueden tener espacios, y distinguen mayúsculas (edad y Edad son distintas). Por convención Python usa snake_case: palabras_separadas_por_guiones.

🚫 Prohibido No puedes usar palabras reservadas como for, if, while, class como nombres de variables.
