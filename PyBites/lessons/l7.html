¿Qué es una Función?

Una función es un bloque de código reutilizable que realiza una tarea específica. Se define con def nombre(): y se llama escribiendo nombre(). Las funciones evitan repetir código y hacen el programa más organizado.

💡 DRY El principio DRY (Don't Repeat Yourself) dice: si escribes el mismo código más de una vez, crea una función.
📨

Parámetros y Argumentos

Las funciones pueden recibir parámetros (variables de entrada) y devolver valores con return. Puedes tener parámetros con valores por defecto: def saludar(nombre='Mundo'):

🎯 *args y **kwargs *args permite pasar cualquier número de argumentos. **kwargs permite pasar argumentos con nombre.
🌐

Ámbito (Scope)

Las variables creadas dentro de una función son locales — solo existen dentro de ella. Las variables fuera son globales. Si necesitas modificar una global desde dentro, usa la palabra global.

⚡ Lambda lambda x: x*2 es una función anónima de una línea. Útil para operaciones simples y rápidas.
