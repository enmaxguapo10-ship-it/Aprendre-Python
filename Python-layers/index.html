Baixar Python Començar

Aprendre Python sense patir

Python és un llenguatge de programació molt utilitzat i fàcil d’aprendre. La seva dificultat depèn de la manera com s’ensenyi. En aquest curs s’utilitzarà una metodologia pràctica, treballant amb exercicis i projectes, i explicant la teoria només quan calgui.

Com aprendrem?

La majoria de les explicacions es faran així:

  • Llegeix

    A cada secció, trobaràs una explicació breu i sovint vídeos.

  • Prova

    Després, practica amb exercicis per consolidar els conceptes.

  • Pista / Solució

    Si et quedes encallat, tindràs pistes o solucions per avançar.

Ajuda

  • Bucles infinits

    Si es crea un bucle infinit, atura el programa ràpidament: tanca la finestra, usa l’Administrador de tasques o reinicia el dispositiu si cal.

  • Problemes amb random

    Assegura’t d’escriure import random al principi i cridar les funcions amb els valors correctes. Guia de random

  • Altres

    Pots consultar al professor, l'Aleix Abril o buscar al llibre de Python
